## 🎤 소개
### 회원관리 시스템 제작을 위한 팀프로젝트
1. member DB 테이블을 이용하여 CRUD 완성
2. 회원가입시 아이디 중복 체크 기능
3. 회원정보 수정 및 삭제시 재확인 기능
4. 주민등록번호 검증 기능
5. 관리자 모드(회원 전체 보기, 회원 등급 수정, 회원 삭제)
